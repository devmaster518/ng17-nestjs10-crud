# ng17-nestjs10-crud

> Full-Stack CRUD Boilerplate built with Angular17 + NestJS10

![App Screenshot](screenshot.png)

**This boilerplate is to setup a new full stack Angular and Nest project with the following features**:

- server side rendering
- tailwindcss
- eslint
- prettier
- docker (not implemented yet)
- file structure

## Angular (Frontend)

```bash
# development
ng serve
# build
ng build

# generate component
ng generate component component-name # (ng g c component-name)
```

Visit http://localhost:4200/

## Nest (Backend)

```bash
# development
$ npm run start
# watch mode
$ npm run start:dev
# production mode
$ npm run start:prod
```

The server is listening on http://localhost:3000/
